## workout-planner
This project is to mimic Workout Interval Timer app on Android.
The purpose is to explore the Jetpack compose and other Jetpack components.

## Tech stacks used:
### 1. Fully Composable
### 2. Jetpack Hilt
### 3. Jetpack Architecture
### 4. Jetpack Room
### 5. Kotlin Coroutine
### 6. WorkManager API
